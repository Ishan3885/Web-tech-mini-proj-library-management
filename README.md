# 📚 Library Book Reservation Portal

## Overview
A simple web application to search and reserve books.

## Features
- 🔍 Search books
- 📚 Reserve books
- 🚫 Availability status

## Tech Stack
- HTML, CSS, JavaScript
- Node.js (Express)

## How to Run
1. Install dependencies:
   npm install

2. Run server:
   node server.js

3. Open browser:
   http://localhost:3000
